Частые вопросы

Что такое Cognitive Core?

5-слойная самохостимая память для AI-агентов. Сохраняет факты, lessons, decisions через MCP-инструмент cognitive_remember, потом находит их через cognitive_recall — и Claude/Cursor/GPT помнят вас между сессиями. Плюс мультиагентные комнаты и приватный git-сервер.

Чем отличается от Notion + GitHub?

Notion — для людей, не для AI. AI не умеет писать туда автоматически и не умеет искать семантически. Здесь — AI пишет/читает сам через MCP, без вашего участия. GitHub — для кода, но не для контекста. Здесь — и git, и память, и инструменты в одном месте.

Можно ли мигрировать данные?

Да. Все ваши L1-события + L3 знания + git-репозитории — экспортируются в JSON / SQL / tarball. Никакого vendor lock-in.

А если я хочу свой сервер (on-premise)?

Enterprise tier — даём docker-compose stack для установки на ваш сервер. Поддержка установки и обновления — включена. Контакт: owner@ии-память.рф.

Как считаются «события»?

Одно событие = один вызов cognitive_remember или один POST /events. Recall (поиск) НЕ считается событием — он лимитируется отдельно (recalls/мин).

Что если я превышу лимит на free-tier?

API вернёт HTTP 429 «quota exceeded» — записи остановятся до следующего дня (для events) или upgrade. Никаких автоматических списаний — это явный сигнал что пора на Pro.

Где находятся серверы?

Россия (для российских клиентов — 152-ФЗ совместимо). Подходит для B2B + B2C. Для Enterprise — можем установить в вашей юрисдикции.

Какие AI-клиенты поддерживаются?

Claude Code, Claude.ai Pro (Custom Connector), Cursor IDE, ChatGPT Plus (Custom GPT), любая платформа с MCP-поддержкой. Подключение за 5 минут через мастер.
