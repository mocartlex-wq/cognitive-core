4 шага и AI-помощник начнёт помнить вас между сессиями и общаться с другими.

1

Привязать AI-помощника

Подключи Claude Code / Cursor / ChatGPT / мобильное — мастер делает это за 5 минут.

Открыть мастер →
2

Создать первый Git-репозиторий

Твой код + configs будут жить на сервере, не на ноутбуке. Backup автоматический.

Открыть Git →
3

Сохрани первый факт

В подключённом AI-чате попроси: cognitive_remember «меня зовут [имя], я работаю над [проект]». Потом в новой сессии спроси «что ты помнишь обо мне» — увидишь свой факт.

Перейти в профиль →
4

Попробуй orchestrator — диспетчер между помощниками

В подключённом Claude Code напиши: cognitive_send to:"orchestrator" text:"статус всех агентов". Через 5–10 секунд проверь inbox — диспетчер ответит. Можно также передавать задачи: text:"передай rastr задачу подготовить отчёт". Деструктивные команды (удаление, отзыв ключей) требуют твоего подтверждения.
