# Code of Conduct

## Кратко

Cognitive Core — open-source проект для людей с разным опытом, языком и культурой.
Будьте вежливы, конструктивны, открыты.

## Что приветствуется

- 🤝 Уважительное общение, даже при разногласиях
- 💡 Конструктивная критика идей (не людей)
- 🌐 Помощь новичкам и начинающим контрибьюторам
- 🎯 Фокус на технических аргументах
- 📖 Терпение к разным уровням английского/русского

## Что недопустимо

- ❌ Личные нападки, оскорбления, унизительные комментарии
- ❌ Дискриминация по любому признаку (раса, пол, ориентация, опыт, язык и т.д.)
- ❌ Sexual harassment, unwanted advances
- ❌ Trolling, deliberate disruption
- ❌ Публикация чужой private information без разрешения
- ❌ Spam, off-topic promotion

## Применимость

Code of Conduct применяется во всех проектных пространствах:
- GitHub Issues, PRs, Discussions
- Чаты (Discord/Slack если будут)
- Email-переписка по проекту
- Конференции и мероприятия от имени проекта

## Сообщение о нарушении

Если вы стали свидетелем или участником нарушения:

1. Если безопасно — мягко указать в самом обсуждении
2. Если ситуация серьёзная — связаться с maintainers privately
3. Email: `conduct@cognitive-core.local` (заменить на реальный при публикации)

Maintainers расследуют, ответят в течение 7 дней. Сообщения остаются конфиденциальными.

## Действия maintainers

В порядке эскалации:
1. **Предупреждение** — частное сообщение с просьбой прекратить
2. **Временный бан** — 1-7 дней без участия в проекте
3. **Постоянный бан** — для repeat-violators или серьёзных нарушений
