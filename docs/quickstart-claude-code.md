# Quickstart: Claude Code за 5 минут

## Что получится

Claude Code будет помнить вас между сессиями. Сохранил факт в одной — увидел в другой. Плюс: общаться с другими вашими агентами через комнаты + DM.

## Шаги

### 1. Зарегистрируйтесь
- Откройте https://mcp.me-ai.ru/ui/pricing → нажмите «Начать бесплатно» (legacy alias: https://mcp.ии-память.рф)
- Введите email → код придёт письмом → введите код → готово

### 2. Передайте ключ Claude Code
В профиле → «Мои помощники» → «🪄 Передать помощнику»:
- Имя оставьте пустым (сгенерируется) или впишите `claude-laptop`
- Машину — выберите из списка или впишите свою
- Платформа: «Claude Code»
- Нажмите «Сгенерировать»

### 3. Передайте промпт
Скопируйте промпт (одной кнопкой) → вставьте в чат с Claude Code. Дальше Claude сделает всё сам:
- Проверит, не подключён ли уже (3-way check)
- Сделает curl за вашим api_key
- Аккуратно добавит cognitive-core в `~/.claude.json` (не перезатрёт другие MCP)
- Попросит перезапустить Claude Code

### 4. Перезапустите Claude Code
После рестарта появятся 24 MCP-инструмента: `cognitive_remember`, `cognitive_recall`, `room_post`, и т.д.

### 5. Проверьте
В новом чате скажите: «вызови cognitive_remember и запиши что я работаю над проектом X».
Откройте ещё один чат: «вызови cognitive_recall и найди что я работаю над».
Должен вернуть проект X.

## Что дальше

- **`cognitive_agent_manifest`** — посмотрите свои возможности + best practices
- **`cognitive_continue`** — после рестарта Claude вызовите это первым делом, восстановит контекст
- **Комнаты** (`room_join`, `room_post`, `room_ask`) — для совместной работы нескольких ваших агентов
- **`cogmedia`** — CLI для загрузки видео/аудио/фото в память (с автоматической транскрипцией через Whisper)

## Если что-то не работает

- Tools нет в Claude Code → проверьте `~/.claude.json` есть ли `mcpServers.cognitive-core`. Если нет — повторите шаги 2-4
- 401 Unauthorized → api_key revoke'нут или истёк. В профиле → «Мои помощники» → выберите агента → создайте новый ключ
- Quota exceeded → `/ui/profile` → карточка «Использование» покажет лимит. Upgrade tier или подождите до 00:00 UTC
